# Metanoia Romanian Baptist Church Website

## Run locally
1. Install Node.js 18+
2. Put your image files in `public/images/`
3. In this folder run:
   npm install
   npm run dev

## Important
The code expects these files in `public/images/`:
- logo.png
- hero.jpg
- congregation-1.jpg
- congregation-2.jpg
- worship-1.jpg
- preaching.jpg
- children.jpg
- ministry.jpg
- gallery-1.jpg
- gallery-2.jpg

## Included
- Bilingual English/Romanian website
- Statement of Faith page
- Pastors page
- Ministries page
- YouTube Streams links
- Google Maps embed
- Facebook page embed
- Give page with Cash App: $MetanoiaTampa
- Prayer request page
